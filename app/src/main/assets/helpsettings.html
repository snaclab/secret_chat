Serval Settings

There are five options on the Settings Screen.

Wifi SettingsWifi Settings allows you to choose the type of WiFi configuration you wish to use currently. This includes allowing Bluetooth and Allowing Wifi. These options can seriously affect your Power consumption.
Accounts ManagementServal uses Accounts to provide initial identity management. More information is available at our Help - Account Management
View LogsA Developer option, to allow you to view issues that are logged.
Mesh TestingThis tests the phone's chipset for Mesh support. This requires Root permissions. More information on issues with Mesh can be found on our Adhoc Page
MeshMS SettingsMeshMS Settings allows you to change configuration for MeshMS
