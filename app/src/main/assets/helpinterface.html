Using the Main Screen

There are 9 buttons on the Main Screen.
There is also the display of your current Phone Number towards the bottom of the screen. This can be changed at any point in the Settings section.

The buttons on the Main Screen

CallMake calls via Serval Mesh
MessagesSend messages via Serval Mesh
ContactsManage contacts, access Address Book
MapsServal Mapping application (if installed)
Share FilesShare files on the Serval Mesh
Share UsShare our software with other people
SettingsSettings for Serval Mesh application
ConnectEnable Serval Mesh processes & connect to Networks
HelpGuides, links, information
AccountsInformation on personal settings
